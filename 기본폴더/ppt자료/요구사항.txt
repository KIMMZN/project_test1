(2) 공지사항 관리

1. 공지사항의 입력과 삭제는 관리자만 할 수 있다.
2. 공지사항 정보: 제목, 카테고리, 내용, 첨부 파일, 시간, 댓글
3. 공지사항 게시판 및 자유 게시판의 글을 삭제 및 수정 할 수 있다


(3) 게시판 (자유 게시판, 공지사항)
1. 게시판 정보: 제목, 카테고리, 내용, 작성자(이름), 첨부 파일, 시간, 댓글
2. 첨부 파일은 3개까지 넣을 수 있다.
3. 게시판 기능: CRUD, paging
4. 게시판에 댓글 기능
5. 공지사항 직원은 조회만 가능
6. 검색 기능 ( 카테고리, 제목 )
7. 댓글 정보: 이름, 부서, 직급, 내용, 날짜 시간
8. 댓글에는 삭제 기능이 있다.

요구사항 정의서 
공지사항 게시판
게시글 작성과 삭제, 수정은 관리자만 할 수 있다.
게시글 작성시 가입된 작성자 이름이 자동 선택된다
게시글은 10개씩 출력되어 볼 수 있다
게시글 작성시 첨부파일은 최대 3개이다
게시글 수정시 글 제목, 내용, 첨부파일 수정을 할수 있다
가입된 사원은 댓글을 작성 할수 있고 본인 댓글은 삭제 할수도 있다
게시글을 검색 할 수 있다

자유게시판
로그인한 사원은 게시글 작성, 수정, 삭제를 할 수 있다.
게시글 작성시 가입된 작성자 이름이 자동 선택된다.
게시글 작성시 첨부파일은 최대 3개이다
게시글 수정시 글 제목, 내용, 첨부파일 수정을 할수 있다
가입된 사원은 댓글을 작성 할수 있고 본인 댓글은 삭제 할수도 있다
게시글을 검색 할 수 있다
